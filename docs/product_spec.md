



# App 功能描述

## 一、产品定位

本 App 是一款面向日语学习者的 **日语听力练习与考试训练应用**。

用户可以按照考试年月选择听力题目，在播放原始音频的同时查看听力原文、题目、选项及中文解释，并通过收藏、循环播放、倍速播放、离线下载等功能反复练习。

主要适合：

- JLPT、大学日语考试或其他日语听力考试学习者
- 希望通过原文对照提高听力的用户
- 需要逐题练习、错题复习和收藏管理的用户

---

# 二、整体页面结构

App 底部包含四个主菜单：

1. **练习**
2. **测试**
3. **收藏**
4. **设置**

页面整体采用黑色背景、白色文字的深色模式。

---

# 三、练习列表页面

## 3.1 页面功能

显示不同年月的听力练习资源，例如：

- 2022年7月
- 2021年12月
- 2021年7月
- 2020年12月
- 2019年12月

每个年月对应一套完整的听力题目。

## 3.2 每套资源显示内容

每个练习卡片包含：

- 考试年月
- 音质信息
- 下载状态
- 本地资源状态
- 可拖动排序图标
- 新内容提示红点

示例：

```text
2022年7月
音质：良好
```

## 3.3 用户操作

用户可以：

- 点击卡片进入对应年月的练习
- 点击下载按钮下载音频和题目数据
- 下载完成后离线使用
- 长按或拖动调整考试资源顺序
- 查看资源是否存在更新

## 3.4 下载功能

建议下载内容包括：

- MP3、M4A 等音频文件
- 题目 JSON 数据
- 日语听力原文
- 中文翻译或解释
- 正确答案
- 音频时间轴数据

下载完成后保存在 App 本地目录中。

# 六、内容模式切换

详情页面提供四种内容模式：

```text
文本｜问题｜文本・问题｜说明文
```

不同模式显示不同内容。

---

## 6.1 文本模式

显示完整的听力原文。

例如：

```text
大学の授業で先生が話しています。

先生は、この授業の一番の目的は何だと言っていますか？

先生：この社会文化概論の授業では……
```

### 功能要求

- 根据音频播放进度高亮当前句子
- 当前播放句子显示为红色
- 其他句子显示为白色
- 用户点击某一句，可以从该句对应时间开始播放
- 支持上下滚动
- 播放时自动滚动到当前句子
- 每句话右侧可以单独收藏

### 高亮方式

当前句子左侧显示红色竖线，文字变成红色。

```text
▌大学の授業で先生が話しています。
```

---

## 6.2 问题模式

仅显示题目和答案选项。

示例：

```text
質問 ＊＊＊＊＊＊＊＊

① さまざまな文化を比較すること
② 社会と文化を分けて考えること
③ 社会と自分を関連させて考えること
④ 建設的な議論の方法を学ぶこと
```

### 功能要求

- 显示题目类型
- 显示隐藏状态的题目文本
- 显示四个答案选项
- 点击选项后记录用户答案
- 选择后显示选中状态
- 提交后显示正确或错误
- 正确答案使用正确标识
- 错误答案使用错误标识
- 保存用户答题记录

### 选项卡片

每个选项为独立卡片：

- 圆角矩形
- 深灰色背景
- 左侧圆形编号
- 支持多行文字
- 点击后改变背景或边框状态

---

## 6.3 文本・问题模式

同时显示：

- 当前题目相关的原文
- 题目
- 答案选项

适合用户在听完音频后，一边查看听力原文，一边完成题目。

### 页面布局

上半部分：

- 当前问题文本
- 播放同步高亮
- 句子收藏按钮

下半部分：

- 问题标题
- 四个答案选项

内容区域可以整体滚动，但底部音频播放器固定。

---

## 6.4 说明文模式

显示中文翻译、题目解析或者听力内容说明。

示例内容：

```text
在大学的舞蹈俱乐部，部长正在和女学生说话。
女学生接下来首先要做的事情是什么？
```

### 功能要求

- 显示中文翻译
- 显示日文原文解释
- 显示正确答案
- 显示错误选项原因
- 支持日文和中文切换
- 支持隐藏或显示翻译

页面可以提供按钮：

```text
中国語翻訳
```

点击后切换：

- 中文翻译
- 日语说明
- 双语对照

# 七、音频播放功能

音频播放器固定在页面底部，切换内容模式时播放不中断。

## 7.1 播放进度

显示：

- 当前播放时间
- 总播放时间
- 播放进度条
- 可拖动的播放滑块

例如：

```text
01:23 ━━━━━━━━━━━━━
```

用户拖动进度条后，音频跳转到对应位置。

---

## 7.2 播放控制

包含以下按钮：

- 倍速
- 上一题或上一段
- 播放、暂停
- 下一题或下一段
- 循环播放

### 倍速设置

点击“倍速”按钮后弹出速度选择：

```text
0.5x
0.75x
1.0x
1.25x
1.5x
1.75x
2.0x
```

系统需要保存用户最近一次选择的倍速。

### 上一段、下一段

可以有两种行为：

1. 切换上一句、下一句
2. 切换上一题、下一题

推荐采用：

- 短按：切换句子
- 长按：切换题目

或者在设置页面中允许用户选择按钮行为。

### 循环播放

支持三种循环模式：

- 不循环
- 当前题目循环
- 当前句子循环

每次点击循环按钮切换模式。

# 八、音频与文字同步

这是整个 App 的核心功能。

## 8.1 实现效果

播放音频时：

1. 获取当前播放毫秒数
2. 查找当前时间属于哪一句文字
3. 将该句文字设为红色
4. 左侧显示红色竖线
5. 自动滚动到该句
6. 点击文字后跳转到对应音频时间

## 8.2 时间轴数据示例

每句话需要保存开始时间和结束时间：

```json
{
  "id": "sentence_001",
  "text": "大学の授業で先生が話しています。",
  "startMs": 0,
  "endMs": 4200
}
```

完整数据：

```json
{
  "questionId": "2022_07_2_7",
  "audioUrl": "audio/2022_07_2_7.mp3",
  "sentences": [
    {
      "id": "s1",
      "text": "大学の授業で先生が話しています。",
      "translation": "大学课堂上，老师正在讲话。",
      "startMs": 0,
      "endMs": 4200
    },
    {
      "id": "s2",
      "text": "先生は、この授業の一番の目的は何だと言っていますか？",
      "translation": "老师说这门课最主要的目的是什么？",
      "startMs": 4200,
      "endMs": 9100
    }
  ]
}
```

# 十六、音频资源采用本地还是网络

最适合采用 **网络下载＋本地缓存的混合方式**。

## 方案一：全部打包在 App 内

优点：

- 完全离线
- 播放稳定
- 不需要服务器

缺点：

- App 安装包会非常大
- 更新题目需要重新发布 App
- 不适合大量年份音频

## 方案二：全部从服务器播放

优点：

- 安装包小
- 内容容易更新

缺点：

- 没有网络时不能使用
- 播放可能卡顿
- 服务器流量成本较高

## 推荐方案：按年月下载到本地

App 首次安装只包含基础数据。

用户点击某个年月的下载按钮后：

1. 从服务器下载题目 JSON
2. 下载音频文件
3. 保存到本地目录
4. 更新数据库中的下载状态
5. 后续直接播放本地音频

这种方式最符合截图中的下载按钮设计。

# 十八、最终效果说明

用户选择一个考试年月后进入听力题目页面。播放音频时，日语原文根据时间轴逐句高亮；用户可以切换查看原文、问题、组合页面以及中文解释，并完成选择题。

底部播放器始终保持显示，支持暂停、进度拖动、倍速、上一段、下一段和循环播放。用户可以收藏题目、下载整套资源并离线练习。

从截图判断，这款 App 的核心不是普通音频播放器，而是：

> **考试题库管理＋音频播放＋文字时间轴同步＋答题系统＋翻译解析＋学习记录。**
